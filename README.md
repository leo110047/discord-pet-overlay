# ODANGO - Discord 桌面寵物

一個 Tauri v2 桌面應用程式，讓你的 Discord 社群互動成為桌面寵物成長的動力。

## 功能特色

- 🐾 **桌面寵物**：可愛的寵物會在螢幕底部走動
- 🐣 **多寵物系統**：從蛋開始孵化，可選擇不同種類的寵物
- 📈 **成長系統**：根據 Discord 活動累積 XP，寵物會成長進化
- 🔗 **配對連動**：使用 Discord ID 和配對碼連結 Discord Bot
- 🖱️ **互動模式**：可拖曳視窗位置和調整寬度
- 🔔 **系統托盤**：最小化到系統托盤，不佔用工作列
- 🆕 **自動更新**：可在設定頁面檢查並下載更新

## 安裝

從 [Releases](https://github.com/mapleleafjack/odango/releases) 下載對應平台的安裝檔：

- **macOS**: `.dmg` 檔案
- **Windows**: `-setup.exe` 檔案

### macOS 注意事項

由於應用程式未經 Apple 簽署，首次開啟時需要：

1. 右鍵點擊應用程式，選擇「打開」
2. 或在終端機執行：
   ```bash
   xattr -cr /Applications/ODANGO.app
   ```

## 使用方式

### 首次配對

1. **啟動應用程式** - 會自動開啟設定視窗

2. **輸入 Discord ID**
   - 在 Discord 設定中開啟「開發者模式」
   - 右鍵點擊自己的頭像，選擇「複製 ID」
   - 將 ID 貼到設定視窗

3. **取得配對碼** - 點擊「發送配對碼」，Bot 會私訊 6 位數配對碼

4. **完成連結** - 輸入配對碼並點擊「連結」

### 互動模式

按住 `Ctrl+O`（macOS: `Cmd+O`）0.5 秒進入互動模式：

- **拖曳移動**：點擊並拖曳視窗
- **調整寬度**：拖曳左右邊緣
- **退出**：再次按住 `Ctrl+O` 0.5 秒

### 設定選項

- **重置視窗位置**：將視窗重置回螢幕底部
- **選擇顯示的寵物**：可選擇要顯示哪些寵物
- **走動開關**：控制寵物是否左右移動
- **速度調整**：調整每隻寵物的移動速度

## 安全聲明

- ✅ Token 儲存在本機，不會上傳到第三方
- ✅ 熱鍵偵測僅用於啟動互動模式，不記錄任何輸入內容
- ❌ 不擷取螢幕畫面

## 疑難排解

| 問題 | 解決方式 |
|------|---------|
| 無法連線 | 確認網路連線正常，稍後再試 |
| 配對碼無效 | 配對碼有效期 10 分鐘，請重新取得 |
| 寵物不顯示 | 點擊「重置視窗位置」或進入互動模式調整 |
| macOS 顯示損壞 | 執行 `xattr -cr /Applications/ODANGO.app` |
